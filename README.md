# SafePay Banking Web App

A simple banking web application built using HTML, CSS, and JavaScript.

## Features
- User login system
- OTP based transaction verification
- Money transfer functionality
- Transaction history tracking
- Separate balance for each user

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage
- Git
- GitHub

## How to Run

1. Download or clone the repository
2. Open index.html in your browser
3. Login using demo credentials
